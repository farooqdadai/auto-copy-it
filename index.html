Key Features

Everything you need for seamless text copying with complete privacy control

Privacy-First Design

Operates entirely locally on your device. No data collection or transmission to external servers. Your privacy is our priority.

Smart Copy Functionality

Automatically copies selected text to clipboard with configurable formats: Plain text, HTML, UPPERCASE, lowercase, Title Case, and Sentence case.

Customizable Settings

Notification system with multiple positions and styles, sound notifications, domain blacklist, and optional copy history storage.

Permission-Based Security

Requires explicit user consent before accessing websites. Clear permission request flow with revocable permissions at any time.

Adjustable Delay

Set copy delay from 0-2000ms to prevent accidental copies. Perfect for precise text selection control.

Domain Control

Blacklist specific websites where you don't want auto-copy functionality. Complete control over where the extension operates.

How It Works

Get started in just 4 simple steps

1

Install Extension

Add Auto Copy It to your Chrome browser from the Chrome Web Store with one click

2

Grant Permissions

Click "Enable Auto-Copy on All Sites" to grant necessary permissions securely

3

Select Text

Select any text on any website you visit - no additional setup required

4

Auto Copy

Text is automatically copied to your clipboard with optional visual notification

Your Privacy Matters

Auto Copy It is designed with privacy-first principles. We believe your data should stay on your device and under your control. No tracking, no analytics, no data collection.

Local Storage Only

All data stored locally using Chrome's secure storage API. Nothing leaves your device.

No Server Communication

Extension operates entirely within your browser. No external connections or data transmission.

User Control

Complete control over all features and data. Enable or disable functionality as needed.

Auto Cleanup

All data automatically removed on uninstall. No traces left behind.
